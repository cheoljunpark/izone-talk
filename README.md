# 프로젝트 이름

IZ\*ONE-Talk

## 소개

이 프로젝트는 HTML과 CSS를 사용하여 카카오톡을 클론코딩 한 것입니다. 이 프로젝트를 통해 HTML과 CSS의 기본 개념과 웹 페이지 레이아웃 기술을 학습하였습니다.

## 사용 기술

- HTML5
- CSS3

## 배포 사이트

- https://cheoljunpark.github.io/izone-talk
